# Alianza F1 Knowledge Agent

Asistente corporativo de inteligencia artificial para consultar y gestionar conocimiento empresarial mediante lenguaje natural utilizando RAG (Retrieval-Augmented Generation).

## Objetivo

Este proyecto nace como parte del Challenge **Alura Agente**, pero está diseñado desde el inicio como la base del futuro asistente corporativo de **Alianza F1**.

El objetivo es desarrollar una solución reutilizable que permita consultar documentación empresarial de forma inteligente mediante IA, facilitando el acceso al conocimiento interno de una organización.

## Visión del proyecto

La evolución prevista del sistema es:

```
Documento
      ↓
Varios documentos
      ↓
Base de conocimiento empresarial
      ↓
Asistente corporativo reutilizable
```

La meta no es construir un chatbot sobre un único documento, sino una plataforma preparada para administrar conocimiento empresarial.

## Características previstas

- Consulta de documentos mediante lenguaje natural.
- Arquitectura basada en RAG.
- Soporte para múltiples documentos.
- Respuestas fundamentadas en las fuentes documentales.
- Arquitectura escalable y reutilizable.
- Preparado para despliegue en Oracle Cloud Infrastructure (OCI).

## Tecnologías previstas

- Python
- LangChain
- Gemini / OpenAI
- Gradio (interfaz inicial)
- Git & GitHub
- Oracle Cloud Infrastructure (OCI)
